#  BigClown Alexa app

This repository contains BigClown Alexa app for connecting between AWS and BigClown.

## Example
```
bch-alexa --certs
```
## Install from GitHub
```
git clone https://github.com/bigclownlabs/bch-alexa.git
```
```
cd bch-alexa
```
```
pip install --editable .
```
